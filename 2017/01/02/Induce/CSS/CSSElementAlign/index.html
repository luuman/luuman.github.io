固定高度绝对定位

使用position:absolute,设置left、top、margin-left、margin-top的属性
这种方法基本浏览器都能够兼容，不足之处就是需要固定宽高。

使用position:fixed,同样设置left、top、margin-left、margin-top的属性

大家都知道的position:fixed,IE是不支持这个属性的
